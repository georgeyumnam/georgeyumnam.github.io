Quick Data Analysis

Small browser tools for everyday measurement files

Drop in raw data files and get an interactive plot in seconds. All processing happens locally in your browser; no data leaves your computer.

PPMS: Electrical Transport

Upload one or more Quantum Design PPMS data files (.dat from the Resistivity or ETO options, or any delimited text file with the same column structure). The tool reads the [Data] block and plots three stacked panels: resistance, phase angle, and the resistance standard deviation. The x axis is chosen automatically for each file: a temperature sweep is plotted against temperature, and a field sweep at fixed temperature is plotted against magnetic field. Curves from every file are overlaid with the file names as legend entries, and clicking a legend entry hides that dataset in all three panels.

Drop PPMS .dat files here, or click to browse

Multiple files supported · .dat, .txt, .csv

Resistance, phase angle, and standard deviation panels will appear here

PPMS: VSM Magnetometry

Upload one or more Quantum Design VSM or MPMS data files (.dat). The tool detects the moment and moment error columns and plots two stacked panels: magnetic moment and its standard error. The x axis is chosen automatically for each file: a temperature sweep (M vs T) is plotted against temperature, and a field sweep (M vs H hysteresis loop) is plotted against magnetic field. Curves from every file are overlaid with the file names as legend entries.

Drop VSM .dat files here, or click to browse
